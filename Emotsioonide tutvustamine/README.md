# Emotsioonide tutvustamine

Veebirakendus kasutab kaamerat ning tuvastab näo põhjal emotsiooni:
- rõõm
- kurbus
- neutraalne
- viha

Rakendus muudab tuvastuse järgi kogu lehe tausta.

## Lisatud 5 valikuvõimalust

1. Figma-stiilis emotsiooni ikoon näo kõrvale (SVG failid kaustas `assets/emotions`).
2. Emotsiooni kindluse protsendiribad.
3. Automaatne kaader emotsiooni vahetusel.
4. Helisignaal emotsiooni muutumisel.
5. Peegelpildi reziim selfie-kaamera jaoks.

## Käivitamine

Ava fail `index.html` brauseris või kasuta lihtsat lokaalset serverit.

> Märkus: Kaamera kasutamiseks võib brauser nõuda https või localhost keskkonda.
